## 📝 About

> Watching the video above and/or accessing the project online will help you understand the explanation!

Collor Gradients is a project inspired by [uiGradients](https://uigradients.com/), being a fictitious company of grandients.<br>
It is also possible to interpret this project as a sales/marketing page for uiGradients, given that, at least so far, it does not have one.<br>
During this project, I managed to improve well mainly my knowledge of HTML and CSS.

### 📌 Some Highlights

- Fully responsive;
- Well-crafted navigation bar;
- Animations with CSS Transition;
- Semantic HTML;

## 📖 Cloning the Project

To clone and run this project on your computer, you just need [Git](https://git-scm.com/) to be previously installed.<br>
After that, in the terminal:

```bash
# Clone this repository with:
> git clone https://github.com/Luk4x/project-collor.git

# Enter the repository with:
> cd project-collor

# Run the project with:
> start index.html # For Windows users
> open index.html # For Linux/Mac users
```
